# **研究背景**

户外广告的投放效果受多重因素共同影响，包括投放规模、时间节点、品牌自身属性等。然而，这些因素对广告效果的实际贡献程度尚不明确——哪些是关键驱动因子？其影响强度如何排序？在预算有限的前提下，又应如何优化资源配置以实现效果最大化？这些问题亟待系统性解答。

为更精准地评估广告效果，本研究选取两个核心业务指标作为分析目标：曝光转化率（衡量用户从看到广告到产生兴趣的效率）与进店贡献率（反映广告对线下到店行为的实际拉动作用）。

基于因果推断方法，本研究旨在识别各影响因子的真实因果效应，量化其相对重要性，并为广告主提供可落地的预算分配策略建议，从而提升户外广告投放的科学性与投资回报率。

# **研究方法**

本研究采用两阶段分析框架：

首先，利用随机森林模型结合SHAP（Shapley Additive Explanations）值，对各类影响因子的重要性进行初步量化与排序，识别出对广告效果具有潜在强关联的关键变量；

在此基础上，进一步引入DML（Double Machine Learning）方法，对筛选出的重要因子与目标指标（曝光转化率、进店贡献率）之间的因果效应进行无偏估计，从而区分相关性与因果性，为资源优化配置提供可靠依据。

## **随机森林**

本研究采用随机森林作为特征重要性初筛的核心模型。该方法通过集成大量去相关化的决策树提升预测稳定性：每棵树基于自助采样的子样本训练，并在节点分裂时仅从随机选取的特征子集中选择最优切分；最终预测结果通过对所有树的输出取平均（回归任务）或投票（分类任务）获得。此设计有效缓解过拟合，对噪声和缺失值具有较强鲁棒性，并可输出各特征的相对重要性评分，为关键影响因子识别提供依据。

## **SHAP（Shapley Additive Explanations）**

为进一步解析变量的作用机制，本研究引入SHAP方法。SHAP 值通过计算每个特征在所有可能特征组合中对预测的边际贡献加权平均，实现对模型输出的公平、可加且一致的归因：

$\phi_i(v)=\sum_{S\subseteq N\setminus\{i\}}\frac{|S|!(n-|S|-1)!}{n!}\left[v(S\cup\{i\})-v(S)\right]$


其在本研究中的价值体现在三方面：
- 方向性识别：明确特征对广告效果是正向促进还是负向抑制；
- 异质性分析：揭示不同品牌、行业或场景下效应的差异；
- 可解释性增强：将模型输出转化为业务可理解的归因逻辑，为后续因果推断提供变量筛选与效应假设支持。

通过“随机森林筛选—SHAP 解释”的两阶段框架，本研究从“哪些因素重要”深入到“它们如何起作用、在何种条件下起作用”，为户外广告资源优化提供实证基础。

## **DML（Double Machine Learning）**

在识别出关键影响因子并解析其作用机制后，为区分相关性与因果性，本研究采用DML方法对核心变量与广告效果指标（曝光转化率、进店贡献率）之间的因果效应进行无偏估计。DML 的核心思想是通过两阶段残差回归消除混杂变量的干扰：（以研究A指标对Y变量的因果效应为例）

$Y=\tau A+g(X)+\varepsilon, \tau为因果效应$

1. 第一阶段：分别用机器学习模型（如随机森林）拟合处理变量（如投放规模）和结果变量（如转化率）对控制变量（如品牌属性、时间趋势等）的条件期望；

$\begin{cases} Y=k(X)+a \\ A=h(X)+ b \end{cases}$
 =>$\begin{cases} \hat{Y}=a=Y-k(X)=\tau A+g(X)+\varepsilon-k(X)=\tau A+\varepsilon' \\ \hat{A}=b=A-h(X) \end{cases}$

2. 第二阶段：基于所得残差构建去偏估计量，通过局部线性回归或工具变量法识别处理变量对结果的因果效应。

$拟合残差a和b：a=xb+y = xA-xh(X) + y$
=>

$Y-k(X)=xA-xh(X)+y$
=>

$Y=xA-xh(X)+k(X)+y$
=>
$x = \tau$


该方法兼具灵活性与稳健性：既能利用非参模型捕捉复杂非线性关系，又能通过正交化设计缓解模型误设带来的偏误。在本研究中，DML 有效解决了户外广告场景中普遍存在的选择性偏差（如高预算往往投向高潜力品牌），从而为资源分配策略提供可靠的因果依据。

- 自然A-IPL流转率\+1%，进店率\+0.04%
- 品牌AIPL规模\+100万，进店率\+0.02%
- 品牌商品均价变化对进店率无显著确定性影响
- 促销期投放，进店率\+0.28%
- 投放城市平均等级\+一档，进店率\+0.05%
- 投放时长\+1天，进店率\+0.01%
- 曝光TA占比10%，进店率\+0.23%
- TA Reach\+1%，进店率\+0.05%
- 人均曝光天数\+1天，进店率\+0.01%
- 订单预算变化对进店率无显著确定性影响

[]

以个护家清为例：
- TA Reach\+1%，进店贡献\+0.20%
- 订单预算\+10万，进店贡献\+0.01%
- cpuv-1毛，进店贡献\+0.82%
- 投放城市数\+1，进店贡献\+0.03%
- 人均曝光天数\+1天，进店贡献\+0.06%
- 自然A-IPL流转率\+1%，进店贡献\+0.23%
- 商品均价-1元，进店贡献\+0.01%
- 品牌AIPL规模\+100万，进店贡献\+0.03%

#### **反直觉结论：为什么部分行业呈现曝光TA占比越高，进店贡献越少的状况？投放城市等级越低，进店贡献越高？**

最重要的原因：进店贡献与曝光人群规模成强正相关。

数据中TA的定义为行业整体的潜客，同一行业的TA总数相同，所以存在曝光人数越少、曝光TA占比越多的情况；针对于进店贡献，曝光人数越多，天然的进店贡献就更高。因此在计算曝光TA占比和进店贡献的关系时，会得到负相关的结论。

[]

曝光人数和投放城市等级之间，也存在一定的规律性，通常曝光的城市越多，意味着规模越大，曝光人数越多，但城市的平均等级会越低，因此会有曝光城市等级越低，进店贡献越高的结论。
